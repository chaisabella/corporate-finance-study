## Exam structure (predictable across years)

3 hours · 100 points · 5–6 questions:

1. **Q1 or Q5:** Capital Structure & Valuation (WACC/APV) — 20–25 pts
2. **Q1 or Q3:** Project Evaluation with FCF — 15–30 pts
3. **Q2:** Options — 15–25 pts
4. **Q3 or Q4:** Swaps — 15–20 pts
5. **Q4 or Q5:** Short Answer (duration / futures / dividends) — 10–15 pts
6. **Q6:** True/False — 10–20 pts

**Time:** ~1.5 minutes per point. **Skip-and-return:** if part (b) stumps you, do (c) — usually independent. **Partial credit is real:** write the formula first, plug numbers second.

## How to study with this

Open `index.html` and work in this order:

1. Read the tutorial at the top of each topic (rose-colored block) — builds the intuition first.
2. Work through each question type's mini-tutorial, formulas, and worked example.
3. Try the drills with solutions hidden. If you can do them, you've got it.
4. For the True/False and short-answer banks, just re-read until the patterns stick.

## Final 24-hour cram strategy

1. Memorize the FCF formula (5 min)
2. Master Topic 1 in order: A → B → C → D (45 min) — the WACC/APV chain
3. Drill Topic 3 binomial methods on the same problem (45 min) — replicating + risk-neutral
4. Memorize put-call parity: $S + P = C + \text{PV}(K)$ (5 min)
5. Memorize the Black-Scholes recipe — not the derivation (30 min)
6. Read the T/F bank twice (20 min)
7. Sleep. Fresh brain > extra hour of cramming.

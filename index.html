Chapter 01
자본구조 & 가치평가
Modigliani-Miller(MM) 이론에서 출발해, 현실의 불완전성이 더해지면서 자본구조가 기업 가치에 영향을 준다는 것을 이해한다.
1.1   Modigliani-Miller — 완전시장에서 자본구조는 무관하다
MM Proposition I (세금 없음)
완전시장(세금 없음, 파산비용 없음, 정보 비대칭 없음)에서는 기업 가치는 자본구조와 무관하다. 즉, 빚을 얼마나 쓰든 회사 전체 가치(V)는 변하지 않는다.
VL = VU
// V_L = 레버리지(부채 있는) 기업 가치, V_U = 무레버리지 기업 가치
MM Proposition II (세금 없음)
빚을 쓸수록 자기자본 요구수익률(r_E)이 높아진다. 이는 주주 위험이 높아지기 때문이다. 빚을 더 쓰면 r_E가 올라가서 WACC는 그대로.
rE = rU + (rU − rD) × D/E
// r_U = 무레버리지 기업 요구수익률 (자산 수익률)
// D/E 올라갈수록 r_E도 올라감 → WACC는 불변
직관

파이(회사 전체 가치)를 어떻게 자르느냐(주식 vs 채권)의 문제일 뿐, 파이 크기 자체는 안 변한다.

1.2   세금 — 이자 세금공제(Interest Tax Shield)

현실에서는 이자 비용이 세금 공제된다. 빚을 쓰면 낼 세금이 줄어들고, 이 절약액만큼 기업 가치가 커진다.

ITS (Interest Tax Shield) = τc × rD × D
// 매기마다 이 금액만큼 세금 절약

// D/V 비율 유지 (= 빚이 기업 가치에 비례해 조정):
PV(ITS) = τc × D/V × VL
// → WACC 방법에 내재되어 있음

// 영구 고정 빚 (= 빚 금액 고정):
PV(ITS) = (τc × rD × D) / rD = τc × D
// → APV 방법에 직접 더함
1.3   가치평가 3가지 방법 — WACC / APV / FTE
방법 ①   WACC (Weighted Average Cost of Capital)
세후 가중평균자본비용으로 FCF를 할인. D/V 비율이 일정하게 유지될 때 적합. ITS가 WACC에 암묵적으로 녹아있다.
rWACC = rE · E/V + rD(1 − τc) · D/V

// 단축 공식 (r_U 이미 알 때):
rWACC = rU − D/V · rD · τc

VL = FCF / rWACC (영구 FCF 가정)
방법 ②   APV (Adjusted Present Value)
먼저 부채 없는 기업 가치(V_U)를 구하고, 부채 사용으로 인한 ITS를 별도로 더한다. 빚 금액이 고정될 때 정확하다.
VL = VU + PV(ITS)

// V_U: 무레버리지 기업 가치 (r_U로 FCF 할인)
VU = FCF / rU

// 영구 고정 빚 D, 이자율 r_D:
PV(ITS) = τc × D

// D/V 유지 빚 (WACC와 같은 가정):
PV(ITS) = τc · rD · D / rU   (r_U로 할인, 위험이 자산과 같음)
⚠ WACC vs APV 선택: 빚이 D/V 비율로 조정되면 WACC. 빚 금액이 고정(영구)이면 APV.
APV(고정 빚)이 WACC(D/V 유지)보다 V_L이 더 크게 나오는 이유: ITS를 r_D로 할인(낮은 위험) vs r_U로 할인(높은 위험) 차이.
방법 ③   FTE (Flow to Equity)
주주에게 귀속되는 레버드 현금흐름을 r_E로 할인해 자기자본 가치(E)를 직접 구한다.
LCF = FCF − rD(1 − τc) · D
// LCF: Levered Cash Flow (세후이자 차감 후 주주 몫)

E = LCF / rE
1.4   Beta Unlever & Relever — 업종 바꿀 때 필수

다른 업종에 투자할 때, 그 업종의 고유 위험(β_U)을 추출한 뒤 내 자본구조에 맞게 재조정한다.

Unlever — 업종 포트폴리오에서 β_U 추출
관측 가능한 β_E와 β_D를 자본구조 가중치로 합산하면 순수 사업 위험 β_U가 나온다.
βU = βE · E/V + βD · D/V
// 업종 포트폴리오의 E/V, D/V, β_E, β_D 사용

// β_D 모를 때: β_D = (r_D − r_f) / E(MRP)
Relever — 내 자본구조로 β_E 재조정
추출한 β_U를 내 회사 D/E로 다시 레버링하여 새 r_E를 구한다.
βEnew = βU + (βU − βD) · D/E
// 내 회사의 D/E 비율 사용

rEnew = rf + βEnew · E(MRP)   (CAPM)
한국 현실 비유 · 삼성바이오로직스
삼성전자가 바이오 사업에 진출할 때, 반도체 베타(β)로 투자를 평가하면 안 되는 이유
삼성전자 β_E ≈ 1.0 (반도체·가전, 시장과 비슷하게 흔들림).
바이오 업종 β_U ≈ 1.6 (임상 실패 시 -40%, 성공 시 +200% 변동성).

반도체 β로 바이오 투자를 평가하면 → 할인율이 너무 낮게 잡힘 → 투자가치 과대평가.
삼성바이오로직스 분식회계 논란도 결국 "자회사를 어떤 할인율로 평가하냐"의 싸움이었다.

올바른 절차: 바이오 업종 포트폴리오에서 β_U 추출 → 삼성의 D/E로 Relever → 바이오용 r_E, WACC 계산.
핵심: 다른 업종 투자 = 그 업종의 β_U를 써야 함. 내 회사 β 그대로 쓰면 틀림.
한국 현실 비유 · 2023 PF 사태
카드론으로 이자 돌려막는 영끌족 — 파산비용이 세금 절약을 초과하는 순간
레버리지(부채)는 이자 세금공제로 절세 이득이 있다. 그러나 2022~23년 금리 급등으로 고레버리지 부동산 시행사들이 연쇄 부도. 태영건설 워크아웃 사태.

최적 자본구조 = ITS의 PV가 파산비용의 PV와 같아지는 지점.
빚이 많을수록: 절세 효과 ↑ 동시에 파산 확률 ↑ & 파산비용(법원 비용, 자산 헐값, 인재 이탈) ↑.

Finance 교과서 결론: 최적 D/V가 존재하며, 이를 넘어서면 오히려 기업 가치가 줄어든다.
최적 자본구조: PV(ITS) = PV(파산비용) 지점
한국 현실 비유 · 전세 사기 / 정보 비대칭
2022~23년 빌라 전세 사기 — 역선택이 시장을 망친다
전세 사기꾼은 자신의 물건이 위험하다는 것을 안다. 보증보험사는 모른다. 위험한 물건만 보험에 몰려 → 보험사 손실 → 시장 붕괴.

Finance에서도 같은 원리: 경영진은 주가가 고평가됐다고 생각할 때 신주를 발행한다. 시장은 이를 눈치채고 주가를 낮춘다.
결과: 기업은 자금조달 순서(Pecking Order)를 따른다 — 내부자금 > 부채 > 신주발행.
정보 비대칭 → Pecking Order: 내부자금 > 부채 > 신주발행
기출 S25 Q1 자본구조 완전 풀이 — r_U · V_U · WACC · APV
업종 포트폴리오: β_E = 1.20, D/V = 0.20, r_D = 5.4%
시장: r_f = 4.2%, E(MRP) = 6.0%, τ_c = 21%
투자 대상: FCF = $51M/년 (영구), 목표 D/V = 0.40, r_D = 6.0%
a
r_U 구하기 (무레버리지 수익률)
업종 포트폴리오의 가중평균 수익률로 r_U를 구한다. 두 가지 경로가 있으며 같은 답이 나온다.
경로 ①: 수익률 가중평균 r_E(업종) = r_f + β_E × E(MRP) = 4.2% + 1.20 × 6.0% = 11.4% r_U = D/V × r_D + E/V × r_E = 0.20 × 5.4% + 0.80 × 11.4% = 1.08% + 9.12%
r_U = 10.2%
경로 ②: 베타 가중평균 → CAPM (검증) β_D = (r_D − r_f) / E(MRP) = (5.4% − 4.2%) / 6.0% = 0.20 β_U = D/V × β_D + E/V × β_E = 0.20 × 0.20 + 0.80 × 1.20 = 0.04 + 0.96 = 1.00 r_U = 4.2% + 1.00 × 6.0%
r_U = 10.2% ✓ 일치
b
V_U — 무레버리지 기업 가치
영구 FCF를 r_U로 할인. 이게 기준값이 된다.
V_U = FCF / r_U = $51M / 0.102
V_U = $500M
c
V_L using WACC — D/V = 0.40 유지할 때
D/V 비율이 기업 가치에 비례해 조정 → WACC 방법 적합.
단축 WACC: r_WACC = r_U − D/V × r_D × τ_c = 10.2% − 0.40 × 6.0% × 0.21 = 10.2% − 0.504%
r_WACC = 9.696%
V_L = FCF / r_WACC = $51M / 0.09696
V_L = $526.0M
검증: r_E → 완전 WACC 경로 r_E = r_U + D/E × (r_U − r_D) [E/V = 0.60, D/E = 0.40/0.60 = 0.667] = 10.2% + 0.667 × (10.2% − 6.0%) = 10.2% + 0.667 × 4.2% = 10.2% + 2.8% = 13.0% WACC = 0.40 × 6.0% × (1−0.21) + 0.60 × 13.0% = 1.896% + 7.80%
= 9.696% ✓
d
부채 금액 D
D = D/V × V_L = 0.40 × $526.0M
D = $210.4M
e
V_L − V_U = PV(ITS) 확인
WACC에서는 ITS가 r_U로 할인된다 (D/V 유지 → ITS 위험 ≈ 자산 위험).
ITS 매기 = D × r_D × τ_c = $210.4M × 6% × 21% = $2.651M PV(ITS) @ r_U = $2.651M / 0.102
= $26.0M
V_L − V_U = $526.0M − $500.0M = $26.0M ✓
f
V_L using APV — 고정 $210M 영구 부채
빚 금액 고정 → ITS가 확정된 부채 계약 → r_D로 할인 → PV(ITS)가 더 커짐 → V_L이 더 크게 나옴.
PV(ITS) = τ_c × D = 0.21 × $210M
= $44.1M
V_L(APV) = V_U + PV(ITS) = $500M + $44.1M
= $544.1M
왜 WACC($526M)보다 큰가? WACC: ITS는 r_U = 10.2%로 할인 (높은 위험) APV: ITS는 r_D = 6.0%로 할인 (낮은 위험, 계약상 확정) → 같은 ITS 금액인데 APV의 PV가 더 큼 → V_L 더 높음
g
왜 APV와 WACC 값이 다른가? (서술)
WACC 가정: 빚이 기업 가치에 비례해 변동 → ITS도 변동 → 자산과 같은 위험 → r_U로 할인.
APV 가정: 빚 금액 고정 → ITS 금액 고정(계약적) → 부채 위험만 → r_D로 할인.

낮은 금리로 할인할수록 PV는 더 커진다. 따라서 APV(고정 빚)의 V_L이 WACC(변동 빚)보다 크다.

Chapter 02
옵션 (Options)
이항모형 → Black-Scholes → Put-Call Parity 순서로 개념을 쌓는다.
2.1   기본 개념
콜옵션 (Call Option)
만기 T에 행사가격 K로 기초자산을 살 권리. 의무 없음. 주가 S_T > K이면 권리 행사, S_T ≤ K이면 포기.
CT = max(ST − K, 0)
풋옵션 (Put Option)
만기 T에 행사가격 K로 기초자산을 팔 권리. S_T < K이면 행사, S_T ≥ K이면 포기.
PT = max(K − ST, 0)
청약 분양권 비유

청약 당첨 시 분양권(콜옵션) 획득. 행사가격 K = 분양가. 입주 시점 시세 = S_T.
시세 > 분양가 → 권리 행사 (입주 or 프리미엄 매도). 시세 < 분양가 → 권리 포기, 손실 = 프리미엄만.
집값 변동성(σ)이 클수록 분양권 가치 ↑ — 오를 가능성이 있기 때문.

2.2   이항모형 (Binomial Model)

주가가 오르거나(u) 내리거나(d) 두 가지만 가정. 두 가지 동치인 방법이 있다.

방법 A — 복제 포트폴리오 (Replicating Portfolio)
주식 Δ주와 채권 B로 옵션의 만기 현금흐름을 그대로 복제한다. 차익거래 불가 원칙에 의해 옵션 현재가격 = 복제 포트폴리오 현재가격.
Δ = Cu − Cd/Su − Sd
// Δ: 헤지비율, 주식 몇 주 살지

B = Cd − Δ · Sd/1 + r
// B < 0 이면 차입 (빌림), B > 0 이면 투자

C = Δ · S + B
방법 B — 위험중립확률 (Risk-Neutral Valuation)
실제 세계의 확률이 아니라, 무위험수익률을 기대수익률로 만드는 가상의 확률 p*를 이용. 더 빠르고, 만기 옵션가치를 p*로 기대값 계산 후 할인.
p* = (1+r) · S − Sd/Su − Sd

C = p* · Cu + (1−p*) · Cd/1 + r

// 두 방법의 결과는 항상 동일
기출 FDMF25 Q2a,b,c 이항모형 — 콜·풋 가격 계산 + Parity 검증
Lewis Inc. 현재 주가 S = $50. 1년 후: S_u = $75 or S_d = $30. r = 4% APR. K = $52.
1
만기 옵션 가치
콜: C_u = max(75 − 52, 0) = $23 C_d = max(30 − 52, 0) = $0 풋: P_u = max(52 − 75, 0) = $0 P_d = max(52 − 30, 0) = $22
2
위험중립확률 p*
p* = [(1.04) × 50 − 30] / (75 − 30) = [52 − 30] / 45 = 22 / 45
p* = 0.4889
3
콜 가격 (a)
C = [p* × C_u + (1−p*) × C_d] / (1+r) = [0.4889 × 23 + 0.5111 × 0] / 1.04 = 11.244 / 1.04
C = $10.81
4
풋 가격 (b) — 위험중립확률 재사용
P = [0.4889 × 0 + 0.5111 × 22] / 1.04 = 11.244 / 1.04
P = $10.81
5
Put-Call Parity 검증 (c)
C − P = S − K/(1+r) 10.81 − 10.81 = 50 − 52/1.04 0.00 = 50 − 50.00
0 = 0 ✓ Parity 성립
2.3   Black-Scholes 모형

이항모형을 무한히 쪼갠 연속시간 극한. 주가가 로그정규분포를 따른다고 가정.

C = S · N(d1) − K · e−rt · N(d2)

d1 = ln(S/K) + (r + σ²/2) · t/σ√t

d2 = d1 − σ√t

// N(x): 표준정규분포 누적확률 (시험에서 테이블 제공)
// N(−x) = 1 − N(x)
각 항의 의미
S · N(d₁): 만기에 콜이 행사될 때 받는 주식 현재가치 (위험 조정됨)
K · e⁻ʳᵗ · N(d₂): 만기에 행사가격을 지불할 현재가치
N(d₂): 위험중립 세계에서 콜이 행사될 확률
기출 S25 Q2a,b Black-Scholes 완전 풀이
S = $50, K = $60, r = 4.15% APR, σ = 35%, t = 1년
1
PV(K) 계산
연속복리로 K를 현재가치로 할인.
PV(K) = K · e^(−r·t) = 60 · e^(−0.0415 × 1) = 60 · 0.9594
= $57.56
2
d₁ 계산
ln(S / PV(K)) = ln(50 / 57.56) = ln(0.8688) = −0.1404 σ²/2 · t = (0.35)² / 2 × 1 = 0.1225/2 = 0.0613 d₁ = (−0.1404 + 0.0613) / (0.35 × √1) = −0.0791 / 0.35
d₁ = −0.23
3
d₂ 계산
d₂ = d₁ − σ√t = −0.23 − 0.35 × 1
d₂ = −0.58
4
N 값 조회 (테이블)
음수 d는 대칭성 N(−x) = 1 − N(x) 이용.
N(d₁) = N(−0.23) = 1 − N(0.23) = 1 − 0.5910 = 0.4090 N(d₂) = N(−0.58) = 1 − N(0.58) = 1 − 0.7190 = 0.2810
5

C = S · N(d₁) − PV(K) · N(d₂) = 50 × 0.4090 − 57.56 × 0.2810 = 20.45 − 16.17
C = $4.28
6
풋 가격 P (Put-Call Parity)
C − P = S − PV(K) P = C − S + PV(K) = 4.28 − 50 + 57.56
P = $11.84

Chapter 03
스왑 (Swaps)
두 당사자가 현금흐름을 교환하는 계약. 금리 스왑과 통화 스왑이 시험의 핵심.
한국 현실 비유 · 2023 저축은행 PF 사태
고정금리 PF 대출 팔고 변동금리로 조달 — 금리 스왑 없이 역마진 폭발
저축은행·증권사: 부동산 PF 대출에 고정 7% 금리 제공 → 고정 수입.
자금 조달: 변동금리 단기 자금 → 금리 오르면 조달 비용 급등.
2022년 한국은행 급격한 금리 인상 → 조달 비용 9%, PF 수입 7% → 역마진 → 부도.

금리 스왑을 했다면: 고정 수취 유지 + 스왑에서 고정 지급·변동 수취 → 순 포지션 = 변동 수취 - 스프레드만. 금리 올라도 조달 비용과 상쇄.
금리 스왑 = 고정↔변동 현금흐름 교환. 금리 위험 헷지 핵심 도구.
3.1   금리 스왑 구조
금리 스왑 (Interest Rate Swap)
두 당사자가 같은 통화, 같은 원금(Notional)에 대해 고정금리와 변동금리를 교환.
원금은 교환하지 않음 (가상의 참조 원금). 이자만 교환.
스왑 시작 시점에는 시장가격으로 거래 → NPV = 0.
// 은행이 고정금리 대출 자산 보유, 변동금리로 헷지하는 예시

[대출에서] 받음: +6.5% × 원금 (고정)
[스왑에서] 지급: −5.0% × 원금 (고정)
[스왑에서] 수취: +v × 원금 (변동)
─────────────────────────────
순 현금흐름: +(v + 1.5%) × 원금 ← 변동금리 자산으로 전환 완료
핵심 원칙

스왑은 두 채권 포지션으로 이해할 수 있다.
고정 수취 스왑 = 고정금리 채권 매수 + 변동금리 채권 매도
금리 오르면 → 고정채권 가격 ↓ → 고정 수취 스왑 가치 ↓

기출 S25 Q4e,f 선도금리 계산 + 스왑 가치 평가
3년 금리 스왑. 은행: 변동 수취, 고정 5.0% 지급. 원금 $130M.
현재(t=0): v₀ = 4.5%, 3년 고정 스왑 = 5.0%
1년 후(t=1): v₁ = 4.95% 확인됨. 새 2년 스왑 고정 = 5.8%.
e
₂f₃ 선도금리 — 기대이론(Expectation Theory)
3년 고정 스왑 = 5.0%는 3년간의 기대 변동금리의 평균. 이를 이용해 ₂f₃를 역추산.
기대이론: (1.05)³ = (1.045) × (1.0495) × (1 + ₂f₃) 좌변: (1.05)³ = 1.15763 우변 앞두 항: 1.045 × 1.0495 = 1.09673 1 + ₂f₃ = 1.15763 / 1.09673 = 1.05552
₂f₃ = 5.55%
f
스왑 현재 가치 — t=1 시점에서 평가
스왑을 "고정 지급 채권 − 변동 수취 채권"으로 이해.
은행: 변동 수취, 고정 지급 → 스왑 가치 = PV(변동 수취) − PV(고정 지급)
PV(고정 지급): 매년 $130M × 5% = $6.5M, 2년 남음, 할인율 5.8% = 6.5/1.058 + (6.5 + 130)/1.058² = 6.144 + 121.949 = $128.093M PV(변동 수취): t=1 직후 = 새로 시작하는 변동금리 채권 → 액면가 = $130.000M 스왑 가치 (은행, 변동 수취 포지션): = PV(변동 수취) − PV(고정 지급) = $130.000M − $128.093M
= +$1.91M (은행 이익)
왜 이익인가?

은행은 5.0% 고정을 지급하는 스왑 계약을 보유 중. 그런데 현재 시장은 5.8% 고정을 요구한다. 은행은 더 낮은 고정금리를 지급하고 있으므로 → 유리한 계약 = 양수 가치.

3.2   통화 스왑 (Currency Swap)
금리 스왑과의 차이
통화 스왑은 원금도 교환한다 (시작 + 만기). 다른 통화의 이자를 교환.
금리 스왑: 원금 교환 없음, 이자만 교환, 같은 통화.
통화 스왑: 시작에 원금 교환, 중간에 이자 교환, 만기에 원금 재교환.
한국 현실 비유 · 원달러 급등
2024~25년 원/달러 1,400원 돌파 — 달러 채권 발행 기업의 환차손 헷지
한국 대기업이 달러 회사채 발행 → 매년 달러 이자 납부 의무.
2024년 12월 한국 계엄 사태 + 트럼프 관세 우려 → 원/달러 1,480원 급등 → 달러 이자 원화 환산 비용 급증.

통화 스왑을 미리 체결했다면: 원화 이자 지급 + 달러 이자 수취.
달러 이자는 스왑에서 받아서 채권자에게 지급. 환율 변동 차단.
통화 스왑 핵심: 원금도 교환(금리 스왑과 차이). 환율 리스크 헷지.

채권 · 선물 · 리스
Duration, Futures pricing, Lease NPV — 계산 집중.
4.1   듀레이션 (Duration)
개념
채권 현금흐름의 가중평균 회수 시간. 금리 변화에 대한 채권 가격 민감도의 지표. 듀레이션이 클수록 금리 변화에 더 크게 반응한다.
D = Σt=1..n t · PV(CFt)/P0

// PV(CF_t) = CF_t / (1+r)^t
// P₀ = 채권 현재 가격 = Σ PV(CF_t)
중요 특성
• 제로쿠폰 채권: D = 만기 (유일하게 D = T)
• 쿠폰 채권: D < 만기
• 금리 ↑ → D 감소, 금리 ↓ → D 증가
• 금리 1% 변화 시 가격 변화 ≈ −D × △r
한국 현실 비유 · 2024 금리 인하
2024년 10월 한국은행 첫 금리 인하 — 국민연금 장기채 평가이익 급등
국민연금은 국내 장기채(10년물, 20년물)를 대규모로 보유.
2024년 한은 기준금리 3.5% → 3.25% 인하 → 장기채 듀레이션 ≈ 8~12년 → 가격 8~12% 상승.
반면 3년 국채 보유자는 가격 2~3%만 상승. 같은 금리 인하인데 장기채 보유자가 훨씬 큰 이익.

역으로 2022년 급격한 금리 인상기엔 장기채 보유 보험사·연기금 평가손실 폭발.
금리 인하 베팅 = 장기채(듀레이션 높은 것) 매수. 듀레이션이 민감도.
기출 FDMF25 Q4b 듀레이션 계산
$1,000 액면 하이일드 채권. 3년간 매년 $400 지급 (원금 별도 상환 없음). YTM = 12.5% APR.
1
채권 가격 계산
PV(CF₁) = 400 / 1.125 = $355.56 PV(CF₂) = 400 / 1.125² = $316.05 PV(CF₃) = 400 / 1.125³ = $281.00 ────────── P₀ =
$952.61
2
t × PV(CF_t) 계산
t=1: 1 × 355.56 = 355.56 t=2: 2 × 316.05 = 632.10 t=3: 3 × 281.00 = 843.00 ───────── 합계: 1,830.66
3
듀레이션
D = 1,830.66 / 952.61
D = 1.92년
금리 1% 상승 시 가격 변화 ≈ −1.92% D = 2.0년인 다른 채권보다 금리에 덜 민감 → 가격 덜 하락
4.2   선물가격 (Futures Pricing)
기본 원리 — 보유 비용 모형 (Cost of Carry)
선물가격 = 현물 매수 후 1기간 보유하는 것과 동일한 비용.
F = S0 · (1 + rf) + PV(보관비용) − PV(편의수익)

// 주식 선물 (배당이 편의수익 역할):
F = S0 · (1 + rf) − PV(배당)

// 편의수익(Convenience Yield) 역추산:
PV(CY) = S0 + PV(보관비) − F / (1+rf)
기출 S25 Q3b 금 편의수익 역추산
금 현물 $3,079.80/oz. 보관비 $25/oz/년 (선불). 1년 선물가격 $3,197.00. r = 4.00% APR.
1
편의수익 공식 정리 후 대입
공식: F/(1+r) = S₀ + PV(보관비) − PV(편의수익) 정리: PV(편의수익) = S₀ + PV(보관비) − F/(1+r) = 3,079.80 + 25 − 3,197.00/1.04 = 3,104.80 − 3,074.04
PV(편의수익) = $30.76/oz

의미: 금을 실물로 보유하는 것이 선물 계약만 보유하는 것보다 연간 $30.76의 추가 가치가 있다. 금융위기·전쟁 시 즉시 사용 가능, 달러 신뢰 하락 시 대안 통화로 기능. 2024~25년 미중 무역전쟁·지정학적 긴장으로 금 편의수익 급증.

4.3   리스 분석 (Lease Analysis)
리스 vs 구매-매각의 NPV
리스의 순현재비용을 구매-매각의 순현재비용과 비교. 할인율은 세후 부채비용 r_D(1−τ_c).
// 리스 NPV (구매 대비 리스의 이득):
NPV리스 = +P0 − PV(리스료 세후) − PV(감가상각 세금절약 포기) − PV(잔존가치)

// P₀: 구매가격 (리스 시 안 내도 됨 → 절약)
// 세후 리스료 = L × (1 − τ_c)
// 감가상각 세금절약 = Dep × τ_c (구매 시에만 받음)
// 할인율 = r_D × (1 − τ_c)
NPV(리스) > 0 → 리스가 구매보다 유리
NPV(리스) < 0 → 구매가 리스보다 유리
기출 S25 Q3c 리스 vs 구매 비교
트럭 구매가 $100,000. 수명 4년. 연 감가상각 $20,000. 4년 후 잔존가치 $20,000 (세금없음).
r_D = 5.0%, τ_c = 21% → 세후 r = 3.95%. 리스료 $20,000/년 (기초 지급, 세전).
1
세후 리스료
L × (1 − τ_c) = $20,000 × (1 − 0.21)
= $15,800/년 (세후)
2
구매 현금흐름 PV (기준선)
감가상각 세금절약: $20,000 × 21% = $4,200/년 (t=1~4) 잔존가치: $20,000 (t=4) PV(구매 이득) = 4,200 × PVAF(4, 3.95%) + 20,000/(1.0395)⁴ = 4,200 × 3.613 + 20,000/1.168 = 15,175 + 17,123 = $32,298 순 구매 비용 = −$100,000 + $32,298
= −$67,702
3
리스 현금흐름 PV
기초 지급 → t=0, 1, 2, 3 에 지급. (기말 지급이면 t=1~4)
PV(리스료) = 15,800 × [1 + PVAF(3, 3.95%)] = 15,800 × [1 + 2.787] = 15,800 × 3.787
= −$59,834
4
결론
리스 비용: $59,834 구매 순비용: $67,702 차이 = $67,702 − $59,834 = $7,868 리스가 $7,868 저렴
→ 리스가 유리
직관: 리스 = 구매 대비 세금 효과 없이 사실상 0% 금리로 자산 이용하는 것. 감가상각 세금절약($4,200×4년) PV보다 구매가($100,000) 차이가 더 큼.
Chapter 05
배당 · 자사주매입 · 지배구조
5.1   MM 배당 무관련성 — 완전시장에서의 결론
MM Dividend Irrelevance (1961)
완전시장에서 배당 정책은 기업 가치와 무관. 주주는 원하는 배당 패턴을 직접 만들 수 있기 때문이다(Homemade Dividend).
// 배당 $D 지급 후:
Pex = Pbefore − D
// 주가는 배당만큼 정확히 하락 → 주주 부는 불변

// 자사주 매입 후:
주가 불변, 주식수 감소 → 주당 가치 상승으로 배당과 동일 효과
현실 세계에서 차이가 생기는 이유: 세금(배당세 vs 양도세 차이), 신호 효과(배당 삭감 = 나쁜 신호), 대리인 문제, 자사주 매입 유연성.
5.2   배당 날짜 순서
공시일(Announcement) → 배당락일(Ex-dividend) → 기준일(Record) → 지급일(Payment)
각 날짜의 의미
공시일: 배당 발표. 예상치 못한 배당이면 주가 반응.
배당락일: 이날 이후 매수하면 배당 못 받음. 주가는 배당금 PV만큼 자동 하락. (t+1 결제 → 배당락일 = 기준일 당일)
기준일: 배당받을 주주 명단 확정.
지급일: 현금 지급.
한국 현실 비유 · 삼성전자
삼성전자가 매년 같은 배당을 발표 — 공시일에 주가가 안 움직이는 이유
시장은 삼성전자가 올해도 작년과 비슷한 배당을 줄 것을 이미 알고 있다. 주가에 이미 반영.
공시일에 예상과 일치하면 → 추가 반응 없음.
2024년 삼성전자가 예상 밖 대규모 자사주 매입을 발표 → 그 날은 주가 급등.

예상된 정보는 이미 가격에 반영 = 효율적 시장 가설(EMH).
배당락일: 주가는 배당금만큼 기계적으로 하락 (시장이 완전히 예상하는 사건).
예상된 배당 → 공시일 주가 반응 없음. 배당락일에만 자동 하락.

Moral Hazard
(도덕적 해이)		 계약 체결 후 행동 변화. 예: 보험 후 부주의 운전. 해결: 모니터링, 공동부담금.
Adverse Selection
(역선택)		 계약 전 정보 비대칭. 예: 위험한 사람만 보험 가입. 해결: 스크리닝(심사). 보험사기 = 도덕적 해이.
한국 현실 비유 · 자동차 보험 사기
자동차 보험 사기 연 1조원 규모 — 도덕적 해이 vs 역선택 구분
도덕적 해이: 보험 가입 후 더 막 운전하거나 의도적 사고 유발 = 계약 후 행동 변화.
해결책: 사고 이력 할증(모니터링), 본인 부담금(공동부담).

역선택: 음주운전 전력자가 보험을 더 열심히 가입 = 계약 전 정보 비대칭.
해결책: 가입 심사 강화, 운전 기록 조회(스크리닝).

2022~23년 빌라 전세 사기도 역선택: 사기꾼(위험한 물건 보유자)이 보증보험을 더 열심히 가입.
도덕적 해이=계약 후 행동 / 역선택=계약 전 정보. 해결책 다름!

WACC

V_L = FCF / r_WACC
D/V 비율 유지. ITS → r_U 할인.
APV
V_L = V_U + τ_c · D
영구 고정 빚. ITS → r_D 할인.
MM Prop II
r_E = r_U + (r_U − r_D) · D/E
빚 늘수록 r_E 증가. WACC 불변.
β Unlever
β_U = β_E·E/V + β_D·D/V
업종 포트폴리오 숫자 사용.
β Relever
β_E = β_U + (β_U−β_D)·D/E
내 회사 D/E 사용 후 CAPM.

d₂ = d₁ − σ√t
N(−x) = 1 − N(x)
Put-Call Parity
